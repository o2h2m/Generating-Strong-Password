# Password Generator - Java

A simple and secure password generator built using **Java**, `SecureRandom`, and `StringBuilder`.

## Features

- Allows the user to:
  - Choose the desired password length
    - Uppercase letters
    - Lowercase letters
    - Numbers
    - Special characters (!@#$%^&* etc.)
- Generates a randomized, secure password
- Simple console-based interface

## Concepts Used
- Java OOP (methods and class structure)
- `SecureRandom` for stronger randomness
- `StringBuilder` for efficient string construction
- Conditional logic and input validation
- printslow function to add a pleasing effect

## Sample Output
Let's assume that the user inputs all the choices and the desired length is 18, the output will be:
pJFjeHN>3y9Kc?om@S
